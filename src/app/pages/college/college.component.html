College Workshops



Electronics



We learn electronics circuit design by playing using virtual platforms. We get different components and connect them on computer. We even test their working and if there are any error computer will guide us.


We start with simple circuits and then slowly transition to micro-controller programming arduino.



Topics

Robotics



This is advance workshop and designed for Computer Science, Engineering and Information Technology Students. The workshop starts with arduino programming, instroduction to various sensors and actuators and connecting them to create a simple robot.


We test the robot using virtual circuits using arduino and then use the real arduino board to create the working robot.



Topics

Thane Road, Gauripada, Bhiwandi, District Thane

Cloud Computing



This is advance workshop and designed for Computer Science, Engineering and Information Technology Students. The workshop starts with simple machine creation, creating a simple web application on it and then exposing it to the outside world. It further introduces the concept of load balancing.


We start with creating a reponsive local web application then moving it to cloud. Running and debugging web application on the cloud. Then accessing the same application using mobile.



Topics

Thane Road, Gauripada, Bhiwandi, District Thane

Internet of Things - IOT



This is advance workshop and designed for Computer Science, Engineering and Information Technology Students. This workshop is conducted after cloud workshop, here we connect the raspberry pi as an IOT device to cloud application.


We configure and connect the raspberry pi as an IOT device, connect to the web application. Change our web application to control the device using our mobile.



Topics

Thane Road, Gauripada, Bhiwandi, District Thane

Home Automation



This is advance workshop and designed for Computer Science, Engineering and Information Technology Students. In this workshop we use the raspberry pi to control the devices in our home.


We develop a simple web application that controls the light and fans using rasberry pi output pins. Then control these using our mobile phone.



Topics
